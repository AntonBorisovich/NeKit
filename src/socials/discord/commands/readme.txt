Здесь команды, которые невозможно унифицировать для каждой соцсети или если команда обязательна для работы этой соцсети. То есть если NeKit будет запущен в режиме Discord и ему будет дана команда settings, то первым делом он проверит, а есть ли settings в папке src/socials/discord/commands, и только потом, если не найдёт таковую, проверит папку src/commands
